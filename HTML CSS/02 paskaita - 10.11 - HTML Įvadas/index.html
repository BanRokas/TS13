Temos kurias mokinsimės


HTML

Hyper Text Markup Language - Hiper teksto žymėjimo kalba skirta surašyti informacijai į puslapį ir ją grupuoti tag'ais.


CSS

Cascading Style Sheets - Kaskadiniai stilių šablonai skirti stilizuoti HTML'ą.


JS

JavaScript - Skirtas pridėti puslapiui funkcionalumą ir/ar reagavimą į vartotojo įvestis.


JS DOM

JavaScript Document Object Model - JavaScript'u keičiamas/redaguojamas/pildomas puslapio kontentas.


TS

TypeScript - Lygiai tas pats JavaScript, tik su griežta rašymo struktūra, kurioje reikia nurodyti duomenų tipus.


React

Galingas svetainių kūrimo įrankis. Jo vienas didžiausių privalumų - niekada nėra perkraunamas puslapis.


NodeJS

Node JavaScript - Skirtas kurti BackEnd'ui, kuriame vykdomi įvairūs veiksmai su duomenimis.

Stilizavimo tag'ai

  • i - Italic - Pakreipia tekstą
  • b - Bold - Pastorina tekstą
  • em - Emphasize - Pakreipia tekstą ir jį pabrėžia(auto reader)
  • strong - Pastorina tekstą ir jį pabrėžia(auto reader)
  • sup - SuperScript / Super(man) - Sumažina tekstą ir pakelia į viršų
  • sub - SubScript - Sumažina tekstą ir nueleidžią į apačią
  • mark - Pažymi tekstą (suteikia lengvą geltoną fono spalvą)
  • u - Underline - Pabraukia žodį (brukšnys po žodžiu)
  • big - Padidina tekstą
  • small - Sumažina tekstą
  • center - Sucentruoja tekstinę informaciją

Tag'ai

  • button - Mygtuko tag'as
  • q - Short quotation - Trumpa citata
  • blockquote - Long quotation - Ilga citata
  • form - Formos tag'as
  • input - Formos įvesties laukelis
  • label - Etiketė skirta nurodyti į nustatytą (pagal id) input'ą
  • select - Pasirinkimo elementas, kurio viduje yra opcijos
  • option - Pasirinkimo elemento viduje esanti opcija
  • textarea - Didelis tekstinis įvesties laukas
  • script - Skirtas rašyti/prijungti JS kodą/failą
  • noscript - Reikalingas parodyti pranešimui jei neveikia javascript variklis pas vartotoją
  • span - Tuščias linijinio tipo elementas skirtas išskirti informaciją
  • fieldset - Skirtas grupuoti susijusius elementus formos viduje
  • legend - Nurodo fieldset'o pavadinimą
  • video - Skirtas atvaizduoti kažkokį vaizdo įrašą
  • iframe - Įkelti kito puslapio turinį arba dalį puslapio turinio
  • table - Lentelės tag'as
  • tr - Table Row - Lentelės eilutė
  • th - Table Heading - Lentelės stulpelio/eilutės pavadinimas
  • td - Table Data - Lentelės duomenys (info vieneto gabaliukas)
  • thead - Table Head - Lentelės galva (viršutinė dalis), kurioje būna stulpelių pavadinimai
  • tbody - Table Body - Lentelės kūnas (vidurinė dalis), kurioje būna lentėlės duomenys
  • tfoot - Table Foot - Lentelės kojos (apatinė dalis), kuri gali atkartoti thead arba atvaizduoti duomenų stulpelio sumą/vidurkį t.t.
  • colgroup - Column Group - Skirtas grupuoti stulpelius (stilizavimui)
  • caption - Skirtas lentelės pavadinimui atvaizduoti
