Tag'ai

  • meta - Nurodyti puslapio informaciją
  • title - Pakeisti tab'o pavadinimą
  • h1 -> h6 - Heading - Pavadinimas
  • p - Paragraph - Paragrafas
  • a - Anchor - Nuoroda
  • img - Image - Paveiksliukas
  • ul - Unordered List - Nerikiuotas sąrašas
  • ol - Ordered List - Rikiuotas sąrašas
  • li - List Item - Rikiuoto/Nerikiuoto sąrašo elementas
  • dl - Description List - Aprašomasis sąrašas
  • dt - Description Term - Aprašamojo sąrašo terminas
  • dd - Description Term Detail - Aprašomojo sąrašo termino paaiškinimas
  • hr - Thematic break (Horizontal Line) - Horizontali linija
  • br - Line break - Eilutės lūžis
  • div - Tuščias blokinio tipo elementas skirtas talpinti informaciją

Atributai

  • lang - Language - html tag'o atributas skirtas nurodyti kokai kalba vyrauja puslapyje.
  • charset - Character Set - meta tag'o atributas skirtas nurodyti puslapiui kokį simbolių rinkinį naudoti.
  • src - Source - Šaltinis skirtas nurodyti, kur randasi failas
  • href - Hyperlink Reference - Hiperlinko nuoroda
  • target - a tag'o atributas skirtas nurodyti, kur atidaryti nuorodą
  • alt - Alternative Text - img tag'o atributas skirtas nurodyti alternatyvų tekstą paveiksliuko neužkrovimo atveju arba automatiniam skaitytuvui
  • width - Plotis - Nurodo elemento plotį (img, video...)
  • height - Aukštis - Nurodo elemento aukštį (img, video...)

Klaviatūros trumpiniai

  • CTRL S - Išsaugoti
  • CTRL C - Kopijuoti
  • CTRL X - Iškirpti
  • CTRL V - Įkiluoti
  • CTRL SPACE - emmetas (info apie elementą)
  • CTRL Z - Pastarojo veiksmo atšaukimas
  • CTRL Z SHIFT / CTRL Y - Pastarojo veiksmo atšaukimo atšaukimas
  • CTRL O - Atidaro failo pasirinkimo langą
  • CTRL F - Atidaro ieškojimo aplinką
  • CTRL Q - Atridaro palečių pasirinkimą
  • WINDOW SPACE / SHIFT ALT - Keičia kalbą
  • dešinys ALT ąčęėįšų - Rašo tuos simbolius originalo kalba
  • SHIFT WINDOW S - Screenshot pasirenkamą vietą
  • CTRL D - Parodo parinkto žodžio duplikatus
  • CTRL L - Pažymi visą eilutę
  • CTRL A - Pažymi viską
  • CTRL ir rodyklės į šonus - Vaikšto per visą žodį
  • CTRL SHIFT F - Paieška failų sistemoje
  • laikant SHIFT ir naudojant vaikščiojimo komandas - parenka tai pro kur eina
  • CTRL B - VSC atidaro šoninį langą
  • CTRL ENTER - Peršoka į naujai sukurtą eilutę
  • ALT ir rodyklė žemyn/aukštyn - Kilnoja eilutes
  • TAB - Atitraukia į dešinį šoną (tai kas pažymėta)
  • SHIFT TAB - Atritraukia į kairį šoną (tai kas pažymėta)
  • ALT ir su pele pažymėti - Sukurti keletą aktyvių kursorių
  • SHIFT ALT ir rodyklė žemyn/aukštyn - Duplikuoja eilutę žemyn/aukštyn
  • ALT Z - Įjungia Word Wrap parinktį
  • CTRL ATL DEL - Kai viskas užlūžta naudoji atidaryti Task Manager(užduočių tvarkytuvas)
  • CTRL U - Grįžta į paskutinę koreguotą eilutę
  • CTRL / - At/Komentuoja pažymėtą eilutę/es
  • ALT TAB - Vaikščioti tarp atidarytų langų
  • WINDOW TAB - Vaikščioti tarp atidarytų langų / kurti naujus ekranus
  • SHIFT ALT F - Automatiškai tabuliuoja visą puslapį
  • WINDOW D / M - Viską minimize'ina (nuleidžia programas į task bar)
  • WINDOW ir rodyklės į šonus - Mažina ir nukelia naudojamą apliakciją į nurodytą ekrano kraštą
  • HOME / HM - Nukelia kursorių į eilutės pradžią
  • END - Nukelia kursorių į eilutės pabaigą
  • PU - PageUP - Nukelia kursorių per ekrano aukštį į viršų
  • PD - PageDOWN - Nukelia kursorių per ekrano aukštį į apačią
  • ALT F4 - Uždaro atidarytą programą
  • PS - PrintScreen - Padaro ekrano/ų nuotrauką
  • CTRL SHIFT WINDOW B - Perkrauna GPU(vaizdo plokštės) driver'ius
  • CTRL T - Naršyklėje atidaro naują tab'ą
  • CTRL SHIFT T - Atidaro uždarytą/us tab'ą/us
