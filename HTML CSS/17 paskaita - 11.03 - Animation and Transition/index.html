Transition

  • transition-property - nurodai kurie stiliai suteikiami vykstant tranzicijai
  • transition-duration - nurodai per kiek laiko įvyks tranzicija
  • transition-timing-function - nurodai kaip tranzicija vyksta per pateiktą laiką
  • transition-delay - nurodai po kiek laiko pradės vykti tranzicija
  • transition - transition-property transition-duration transition-timing-function transition-delay

Timing functions

  • linear
  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • step-start
  • step-end
  • cubic-bezier

Transform

Scale

Scale: scaleX; scaleY; scaleZ; scale; scale3d;
Scale nusako kaip keičiasi elemento dydis. Default dydis yra 1.

Translate

Translate: translateX; translateY; translateZ; translate; translate3d;
Translate nusako į kurią pusę ir kokiu atstumu elementas pajudės.

Rotate

Rotate: rotateX; rotateY; rotateZ; rotate; rotate3d;
Rotate nusako kurioje ašyje ir kiek elementas suksis.

Skew

Skew: skewX; skewY; skew;
Skew nusako kaip yra ištempiami elemento kampai.

Animation

  • animation-duration - animacijos trukmė
  • animation-timing-function - animacijos tekmė per pateiktą laiką
  • animation-delay - animacijos užlaikymas
  • animation-iteration-count - animacijos vykdymo kartai
  • animation-direction - animacijos vykdymo kryptis
  • animation-fill-mode - nusako kokia bus elemento būsena prieš ir/ar po animacijos
  • animation-play-state - animacijos būsena
  • animation-name - animacijos vardas
  • animation - apjungia visus animacijos nustatymus

KeyFrames

@keyframes animation-name { ... }
Animacijos frame'ai aprašomi nuo(from) - iki(to) arba procentaliai nuo(0%) - iki(100%).
Animacijos laikas yra skirtomas frame'ams pagal jų procentalius nurodymus.
