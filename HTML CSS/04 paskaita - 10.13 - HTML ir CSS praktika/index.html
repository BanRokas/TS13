Semantiniai tag'ai

  • header - viršutinė puslapio dalis
  • nav - navigation (menu)
  • main - pagrindinė puslapio dalis (content)
  • article - content dalis (didesnė)
  • section - content dalis (didesnė)
  • aside - šoninė sekcija (dažnai naudojama reklamom arba extra menu)
  • footer - apatinė puslapio dalis

Pozicijos

Static

Statinė pozicija nesuteikia jokių stilių ir neįgalina kitų stilių naudojimo. Yra default nustatymas.

Relative

Reliatyvioji pozicija nesuteikia jokių stilių, bet įgalina top/left/right/bottom ir z-index stilių naudojimą. Top/left/right/bottom atitraukia elementą per nurodytą atstumą, nuo nurodytos kraštinės nuo elemento pradinės pozicijos. Nurodo absolute elemento ribas.

Absolute

Absoliučioji pozicija išeina iš puslapio tėkmės ir įgalina top/left/right/bottom ir z-index stilių naudojimą. Top/left/right/bottom atitraukia elementą per nurodytą atstumą nuo artimiausio protėvio, kuris turi relative poziciją. Užima tiek vietos, kiek reikalauja jo content, jeigu nėra nustatytas dydis.
