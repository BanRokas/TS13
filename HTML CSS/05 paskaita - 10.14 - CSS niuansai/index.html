Tėviniai ir vaikiniai elementai

Elementas esantis kažkokio elemento viduje yra vaikinis elementas, tam elementui, kurio viduje jis yra.
Elementas turintis savo viduje kitus elementą, yra tėvinis elementas, tam kitam elementams, kuris yra jo viduje.

Paveldėjimas

Iš protėvinio elemento, vaikiniai elementai paveldi tam tikrus stilius. Paveldimi stiliai dažniausiai būna kažkas paprasto kaip: teksto spalva, šriftas, dydis ir pns... Nepaveldimi yra dydžio, display, fono, rėmelių ir pns. nustatymai.

  • antras

Stilių prioritetai

  1. Inline Style
  2. ID
  3. Class
  4. Element Selector
  5. CSS order (in CSS code)
  6. Element default style
  7. Inheritence

Esant dviem selektoriam, kurie nusako to paties elemento vienodus stilius yra atsižvelgiama į tų selektorių prioritetus. Aukščiau esantys selektoriai, turi didesnį prioritetą (žiūrėti į sąrašą).
Esant dviem vienodo prioriteto selektoriams (keičiant vienodą stilių) yra atsižvelgiama į CSS eiliškumą (kuris CSS parašytas vėliau, tas ir bus taikomas).

JEIGU ĮMANOMA - NENAUDOTI!!! !important prierašas prie stiliau ignoruoja visus prioritetus ir suteikia tą stilių ir niekas jo pakeist negali apart kito !important, kuris turėtų aukštesnį prioritetą.

Browser Default Styles

Tag'ai gali turėti savitus stilius, kurie jiem yra suteikiami pačių naršyklių.
Kartais skirtingos naršyklės turi minimalistiškai skirtingus numatytuosius stilius. Tai gali paveikti svetainės išvaizdą.
Kartais numatytieji stiliai pagadina svetainės vaizdą arba apsunkina programuotojo darbą.

Yra keletas būdų spręsti nenorimų numatytų stilių sukeliamas problemas.
Vienas iš būdų yra nuimti visus numatytuosius stilius nuo savo svetainėje naudojamų tag'ų yra naudoti Reset CSS.
Kitas (plačiau naudojamas) būdas yra normalizuoti numatytuosius stilius tarp skirtingų naršyklių ir dalį jų panaikinti/pakeisti naudojant Normalize CSS.
