Media Queries

Media queries naudojami kai norime, kad ekranui esant specifiniam pločiui yra naudojami specifiniai stiliai. (maži ekranai, turi kitokius stilius, nei dideli ekranai).

@media extra (min/max-width: 999px) { styles }

Galima nurodyti, kad tam tikri stiliai galiotų tik tam tikrose ekrano pločio ribose:
@media extra (min-width: 500px) and (max-width: 600px) { ... }

Breakpoints

Ekrano lūžio taškai nurodytuose ekrano pločiuose. Lūžio taškų pločius nusirodome kokius tik norime. Minimalistiškai skirstoma į 3 ekranus: mobile, tablet, desktop.
