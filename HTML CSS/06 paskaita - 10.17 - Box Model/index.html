Box Model

Content

Elemento turinys tai ekrane atvaizduojama informacija vartotojui. Jinai gali būti tekstas, paveiksliukas, fonas... Turinio dydį nusako turinio kiekis / dydis (nustatytas per CSS) / display.

Padding

Pamušalas yra atstumas tarp elemento turinio ir rėmelių, kuris dažniausiai naudojamas tam, kad elemento turinys nebūtų prilipęs prie savo kraštų.

CSS rašymo būdai

  • padding: top right bottom left;
  • padding: top right/left bottom;
  • padding: top/bottom right/left;
  • padding: top/right/bottom/left;
  • padding-top: top;
  • padding-right: right;
  • padding-bottom: bottom;
  • padding-left: left;

Border

Rėmelis yra tarp Padding ir Margin. Rėmelis yra paskutinė atvaizduojama elemento dalis (fonas). Kitaip pasakius - rėmelis yra matomo elemento kraštas.

CSS rašymo būdai

  • border: top/right/bottom/left;
  • border-top: width type color;
  • border-right: width type color;
  • border-bottom: width type color;
  • border-left: width type color;
  • border-width: top/right/bottom/left;
  • border-style: top/right/bottom/left;
  • border-color: top/right/bottom/left;
  • border-radius: topLeft|topRight|bottomRight|bottomLeft;
  • border-right-color: rightColor;
  • ...
  • border-[kraštinė]-[nustatymas]: nustatymoParinktis;

Margin

Paraštė yra atstumas nuo elemento rėmelio iki aplinkinių elementų, kuris dažniausiai naudojamas tam, kad elementą patrauktume nuo kitų elementų.

CSS rašymo būdai

  • margin: top right bottom left;
  • margin: top right/left bottom;
  • margin: top/bottom right/left;
  • margin: top/right/bottom/left;
  • margin-top: top;
  • margin-right: right;
  • margin-bottom: bottom;
  • margin-left: left;
