CSS kintamieji

CSS'e kintamieji naudojami tam, kad nereiktų atsiminti specifinių plačiai naudojamų nustatymų svetainėje. Vietoj to, kad atsiminti kokio dydžio tekstas turi būti skirtas specifinėms puslapio dalims, galima susikurti kintamuosius su logiškais pavadinimais, kuriems yra priskirti norimi teksto dydžiai ir tuomet lieka atsiminti tik pačius pavadinimus, o ne dydžius.
Labai naudinga turėti kintamuosius norint globaliai pakeisti stilius daugeliui elementų. Daugelis elementų naudoja vieną kintamąjį ir tuomet pakeitus tą kintamąjį - pasikeičia visi elementai.

CSS kintamuosius aprašome CSS (dažniausiai) root'o dalyje (:root{}) ir norėdami kurti naują kintamąjį turime rašyti du brūkšnelius ir kintamojo pavadinima, dvitaškį ir reikšmę (--kintamojo-pavadinimas: reikšmė).

CSS kintamuosius naudojame kreipdamiesi į juos su var (variable) kreipiniu ir skliausteliuose nurodomu kintamojo pavadinimu (--kintamojo-pavadinimas).
