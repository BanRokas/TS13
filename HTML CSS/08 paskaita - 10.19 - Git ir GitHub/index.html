Terminal commands

  • cd - change directory
  • ls - list files in current directory (terminal / gitBash / ...)
  • dir - list files in current directory (cmd)
  • mkdir - make directory - sukuria naują aplankalą
  • touch - sukurti naują failą
  • cat - sukurti ir redaguoti naują failą arba peržiūrėti esamą
  • clear - išvalo komandinę eilutę
  • rm - ištrinti failą / aplankalą (-r komanda trina rekursyviai)
  • CTRL C - atšaukia vykdomą veiksmą
  • ARROW UP - komandinėje eilutėje pateikia jūsų paskutinę komandą

Git setUp

  1. git config --global user.name "Tavo vardas"
  2. git config --global user.email "pastasKuriuoJungeisiPrieGitHub@gmail.com"
  3. git config --global core.editor "code"
  4. git config --global core.autocrlf true (vietoje "true" - "input" jei linux/mac)
  5. git config --global -e - atidarys ir parodys info per editoriu
  6. git config --help

Git commands

  • init - Sukuria naują git'o repozitoriją
  • status - Parodo git'o repozitorijos statusą
  • add - Prideda nurodytą failą/failus prie commit'ui paruoštų failų (stage'ina)
  • commit - Sukuria kodo versiją
  • restore - Panaikina esamus pakitimus failuose / išima iš commit'ui paruoštų failų sąrašo (unstage'ina) jeigu yra prierašas --staged
